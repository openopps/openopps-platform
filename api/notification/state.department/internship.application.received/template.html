What happens next?

The U.S. Department of State Student Internship Program will:

  1. Review all applications.
    They will begin to review applications after the application period closes on <%- moment(cycle.applyEndDate).format('MMMM DD, YYYY') %>.
  2. Set up interviews.
    They will contact you if you're selected for an interview. However, some offices don't conduct interviews.
  3. Send offer emails.
    They will send you an email with a conditional offer, if you're selected for an internship.

Can I update my application?

Yes, you can update your application. You must make all updates before 11:59 p.m. EST on <%- moment(cycle.applyEndDate).format('MMMM DD, YYYY') %>. Don't forget to click submit.

What happens if I'm not selected?

After all internships are filled, you will receive an email if you're not selected.
